# cherkashchenko_projekt

Môj súbor má veľký objem, preto som ho nahrala na Google Drive
https://drive.google.com/file/d/1ftMo_tLYVpFsDNyRaaRwJXdQ0E5dQ8fZ/view?usp=sharing

Pred použitím panela je potrebné spustiť skripty v správnom poradí. Najskôr je potrebné spustiť skript scatter_trees.py, ktorý zabezpečuje automatické rozmiestňovanie objektov v scéne. Následne sa spustí skript growth_animation.py, ktorý vytvára animáciu rastu rastlín. Ako posledný sa spustí skript panel.py, ktorý vytvorí ovládací panel Forest Controls v bočnom menu Blenderu.
